# 💰 Expense Tracker

A full-stack expense tracking web app built with Django and MySQL.

## 🚀 Live Demo
[Link coming soon after deployment]

## 🛠️ Tech Stack
- **Backend:** Python 3.11, Django 5.2
- **Database:** MySQL 8.0
- **Frontend:** Bootstrap 5, Chart.js, Bootstrap Icons
- **Deployment:** Render

## ✨ Features
- User registration and login
- Add, edit, delete expenses
- Category management
- Filter by category and date range
- Monthly budget alerts (warns at 90%)
- Analytics dashboard (pie chart + bar chart)
- Responsive Bootstrap UI
